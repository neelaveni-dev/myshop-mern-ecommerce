# MyShop - MERN E-Commerce Web Application

## Overview

MyShop is a Full Stack E-Commerce Web Application built using the MERN Stack (MongoDB, Express.js, React.js, Node.js).

The application allows users to browse products, search products, view product details, add products to cart, place orders, register, and login.

---

## Features

- User Registration
- User Login Authentication
- Product Listing
- Category-Based Products
- Product Details Page
- Search Products
- Add To Cart
- Update Cart Quantity
- Remove From Cart
- Checkout Page
- Order Success Page
- Responsive User Interface

## Installation

### Clone Repository

```bash
git clone https://github.com/yourusername/myshop-mern-ecommerce.git
```

### Install Frontend

```bash
cd client
npm install
npm run dev
```

### Install Backend

```bash
cd server
npm install
npm start
```

---

## Environment Variables

Create a .env file inside server folder.

```env
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
PORT=5000
```
